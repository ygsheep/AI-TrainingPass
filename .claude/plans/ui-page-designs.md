# UI Page Designs - TrainingPass (Flutter)

## HomePage (首页)

### 设计风格
- **风格**: Swiss Modernism / Minimalism
- **特点**: 干净的边框、无阴影、专注内容

### 布局结构

```
┌─────────────────────────────────────────────┐
│  [Logo] TrainingPass        [⚙️ 设置]      │  <- AppBar
├─────────────────────────────────────────────┤
│                                             │
│  欢迎回来                                   │  <- 欢迎标题
│  准备好开始学习了吗？                        │
│                                             │
│  ┌─────────────────────────────────────┐   │
│  │ [▶] 开始练习                        │   │  <- 主卡片
│  │ 随时答题，巩固知识                   │   │     (蓝色边框)
│  └─────────────────────────────────────┘   │
│                                             │
│  ┌─────────────────┐  ┌─────────────────┐ │
│  │ [?] 模拟考试     │  │ [🔖] 错题本     │ │  <- 次要卡片
│  │ 限时测试        │  │ 复习错题        │ │     (2x2网格)
│  └─────────────────┘  └─────────────────┘ │
│  ┌─────────────────┐  ┌─────────────────┐ │
│  │ [🕐] 历史记录   │  │ [☁️] 题库更新    │ │
│  │ 查看答题历史    │  │ 导入最新题库     │ │
│  └─────────────────┘  └─────────────────┘ │
│                                             │
├─────────────────────────────────────────────┤
│  [🏠] [📖] [📝] [📚] [⚙️]                 │  <- BottomNav
└─────────────────────────────────────────────┘
```

### 响应式布局
| 屏幕尺寸 | 列数 | 卡片比例 |
|----------|------|----------|
| Mobile (<768px) | 1 | 16:9 |
| Tablet (768-1024px) | 2 | 1:1 |
| Desktop (≥1024px) | 3 | 1:1 |

### 组件文件
- `lib/presentation/pages/home/home_page.dart` - 主页面
- `lib/presentation/widgets/cards/home_card.dart` - 卡片组件

### 颜色规范
| 元素 | 浅色模式 | 深色模式 |
|------|----------|----------|
| 背景 | #FFFFFF | #0A0A0A |
| 卡片背景 | #FFFFFF | #242424 |
| 边框(普通) | #E5E5E5 | #333333 |
| 边框(主卡片) | #1976D2 | #1976D2 |
| 图标背景 | rgba(24,118,210,0.1) | rgba(66,165,245,0.1) |

### 触摸尺寸
- 按钮: 48px
- 图标按钮: 48px
- 卡片内边距: 16-24px
